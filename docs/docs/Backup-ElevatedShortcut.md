# Backup-ElevatedShortcut

## SYNOPSIS
Exports the RunAss shortcuts, to a zip file

## SYNTAX

```
Backup-ElevatedShortcut [[-ExportPath] <DirectoryInfo>] [<CommonParameters>]
```

## DESCRIPTION
Exports the RunAss shortcuts, to a zip file

## EXAMPLES

### EXAMPLE 1
```
Backup-ElevatedShortcut -ExportPath c:\temp
```
