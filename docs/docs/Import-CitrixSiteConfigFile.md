# Import-CitrixSiteConfigFile

## SYNOPSIS

Import-CitrixSiteConfigFile [[-CitrixSiteConfigFilePath] <FileInfo>] [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### EXAMPLE 1
```
Import-CitrixSiteConfigFile -CitrixSiteConfigFilePath c:\temp\CTXSiteConfig.json
