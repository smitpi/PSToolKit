# Install-BGInfo

## SYNOPSIS
Install and auto runs bginfo at startup.

## SYNTAX

```
Install-BGInfo [-RunBGInfo] [<CommonParameters>]
```

## DESCRIPTION
Install and auto runs bginfo at startup.

## EXAMPLES

### EXAMPLE 1
```
Install-BGInfo -RunBGInfo
```
