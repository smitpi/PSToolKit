# Export-PSGallery

## SYNOPSIS
Export details of all modules and scripts on psgallery to excel

## SYNTAX

```
Export-PSGallery [[-ReportPath] <String>] [<CommonParameters>]
```

## DESCRIPTION
Export details of all modules and scripts on psgallery to excel

## EXAMPLES

### EXAMPLE 1
```
Export-PSGallery -ReportPath c:\temp
```
