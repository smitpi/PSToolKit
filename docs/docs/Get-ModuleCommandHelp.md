# Get-ModuleCommandHelp

## SYNOPSIS
Show help file of a command in a specified module

## SYNTAX

```
Get-ModuleCommandHelp [<CommonParameters>]
```

## DESCRIPTION
Show help file of a command in a specified module

## EXAMPLES

### EXAMPLE 1
```
Get-ModuleCommandHelp
```
