# Set-TempFolder

## SYNOPSIS

Set all the temp environmental variables to c:\temp

## SYNTAX

### __AllParameterSets

```
Set-TempFolder [<CommonParameters>]
```

## DESCRIPTION

Set all the temp environmental variables to c:\temp


## EXAMPLES

### Example 1: EXAMPLE 1

```
Set-TempFolder
```
