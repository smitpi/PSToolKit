# Set-PSToolKitSystemSetting

## SYNOPSIS

Set multiple settings on desktop or server

## SYNTAX

### __AllParameterSets

```
Set-PSToolKitSystemSetting [-DisableFirewall] [-DisableInternetExplorerESC] [-DisableIPV6] [-DisableServerManager] [-EnableRDP] [-ExecutionPolicy] [-FileExplorerSettings] [-IntranetZone] [-IntranetZoneIPRange] [-PerformReboot] [-PSTrustedHosts] [-RemoveDefaultApps] [-RunAll] [-SetPhotoViewer] [-SystemDefaults] [<CommonParameters>]
```

## DESCRIPTION

Set multiple settings on desktop or server


## EXAMPLES

### Example 1: EXAMPLE 1

```
Set-PSToolKitSystemSettings -RunAll
```
