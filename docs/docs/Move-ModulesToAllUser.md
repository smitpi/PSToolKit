# Move-ModulesToAllUser

## SYNOPSIS
Move modules from current user to all users

## SYNTAX

```
Move-ModulesToAllUser [<CommonParameters>]
```

## DESCRIPTION
Move modules from current user to all users

## EXAMPLES

### EXAMPLE 1
```
Move-ModulesToAllUser -Export HTML -ReportPath C:\temp
```
