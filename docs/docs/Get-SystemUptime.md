# Get-SystemUptime

## SYNOPSIS
Calculates the uptime of a system

## SYNTAX

```
Get-SystemUptime [[-ComputerName] <String[]>] [[-Credential] <PSCredential>] [<CommonParameters>]
```

## DESCRIPTION
Calculates the uptime of a system

## EXAMPLES

### EXAMPLE 1
```
Get-SystemUptime -ComputerName Neptune
```
