# New-PSGenericList

## SYNOPSIS
Creates a .net list object

## SYNTAX

```
New-PSGenericList [[-Type] <Type>] [-Values <Object[]>] [<CommonParameters>]
```

## DESCRIPTION
Creates a .net list object

## EXAMPLES

### EXAMPLE 1
```
$list = New-GenericList -Type string -Values 'blah','two','one'
```
