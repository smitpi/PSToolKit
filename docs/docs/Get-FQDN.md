# Get-FQDN

## SYNOPSIS
Get FQDN for a device, and checks if it is online

## SYNTAX

```
Get-FQDN [-ComputerName] <String> [<CommonParameters>]
```

## DESCRIPTION
Get FQDN for a device, and checks if it is online

## EXAMPLES

### EXAMPLE 1
```
get-FQDN -ComputerName Neptune
```
