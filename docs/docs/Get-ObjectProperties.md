# Get-ObjectProperties

## SYNOPSIS
Cast an array or PSObject and display it in list view

## SYNTAX

```
Get-ObjectProperties [[-Data] <Object[]>] [<CommonParameters>]
```

## DESCRIPTION
Cast an array or PSObject and display it in list view

## EXAMPLES

### EXAMPLE 1
```
Get-ObjectProperties -data $data
```
