# Test-SystemOnline

## SYNOPSIS
Does basic checks for connecting to a remote device

## SYNTAX

```
Test-SystemOnline [[-ComputerName] <String[]>] [[-Credential] <PSCredential>] [<CommonParameters>]
```

## DESCRIPTION
Does basic checks for connecting to a remote device

## EXAMPLES

### EXAMPLE 1
```
Test-SystemOnline -ComputerName $ListOfBoxes -Credential $User
```
