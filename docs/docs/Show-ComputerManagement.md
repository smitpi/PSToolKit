# Show-ComputerManagement

## SYNOPSIS

Opens the Computer Management of the system or remote system

## SYNTAX

### __AllParameterSets

```
Show-ComputerManagement [[-ComputerName <String[]>]] [<CommonParameters>]
```

## DESCRIPTION

Opens the Computer Management of the system or remote system


## EXAMPLES

### Example 1: EXAMPLE 1

```
Show-ComputerManagement -ComputerName neptune
```
